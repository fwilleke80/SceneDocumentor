Ideas for improvement

Just some thoughts about additional features that may be implemented.

Better templates

At the moment, there's a header template and a footer template. The middle part of the document is created from hard coded HTML that can be found in ccreatedocument.cpp.

It would be nice to have only one template that includes something like iteration placeholders to mark the place where the object and tag listing, as well as the comment texts will be inserted.

Multiple templates

How about subfolders in the template folder, to have several sets of different templates? The dialog window could be extended by a listview or dropdown box to select the desired template.

Better image generation

At the moment, the image is rendered in the background, invisible to the user. The status bar is not being correctly updated. It would be a good idea to process the rendering in the Picture Viewer.

It would also be nice to have different placeholders, e.g. to include several different views (perspective, top, front, etc) into the document.

Information Object

How about an object that holds the Title, Author and Free Text information, that must currently be specified in the dialog?

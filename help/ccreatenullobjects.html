The "Create Null Objects" Command

This command will trigger the "Create Null Object" function for each Comment Tag in the scene. The only difference is that the Null Objects are not created above the objects that carry a Comment Tag. They are all put into a group called "Information" on top of the hierarchy.

After calling this command, you may remove the Comment Tags from your scene and pass it on to someone who doesn't have the SceneDocumentor plugin installed.

HTML Templates

When you generate an HTML expos√© of your scene, the resulting HTML document is made from template files. By editing them you can control the look of the generated documents. The template files are basically standard HTML files (of course, you can use CSS) that contain some placeholders.

You can find the template files in the folder plugins\SceneDocumentor\template.

Placeholders

The placeholders will be replaced by certain text contents during document generation. Here's a list of all available placeholders:

%DOC_TITLE%

Inserts the document title, as specified in the "Create HTML Document" dialog.

%DOC_FILENAME%

Inserts the filename of the scene.

%DOC_INTROTEXT%

Inserts the text specified in the big free text box in the dialog.

%DOC_AUTHOR%

Inserts the author name, as specified in the dialog.

%DOC_GENERATOR%

Inserts the name and version of the SceneDocumentor plugin.

%DOC_IMAGE%

Inserts a rendered image of the scene. Details can be specified in the dialog.

%DOC_TIME%

Inserts the current time in HH:MM:SS format.

%DOC_DATE%

Inserts the current date in YYYY-MM-SS format.

%DOC_YEAR%

Inserts only the current year (good for copyright information).

CSS classes

The names of objects and tags, as well as the comment text are provided with special CSS classes. If you want, you can include CSS information in the header template to further style the document.
